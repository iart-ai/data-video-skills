# Data Video Skills

> Data video / animated infographic skills for AI coding agents — turn a CSV into a chart that moves, with exact, accurate numbers on every frame.

## Install

```bash
npx skills add iart-ai/data-video-skills
```

Works with Claude Code, Cursor, Codex, GitHub Copilot, and 40+ agents.

## What's included

| Skill | What it does |
|---|---|
| [data-video](./data-video) | Turn a dataset into an animated chart — bar chart race, line/area reveal, count-up tickers, and batch render one template across many CSVs. |
| [animated-infographic](./animated-infographic) | Build a designed infographic — icons, key numbers, connectors — and reveal it in a staggered cascade. |
| [presentation-video](./presentation-video) | Rebuild a slide deck as a narrated, auto-advancing video with build reveals timed to the voiceover. |

## When it activates

- You ask to make a bar chart race, turn a CSV into a video, or animate a chart over time.
- You ask for an animated counter, number ticker, or growing line/area chart.
- You ask to build an animated infographic, stat-card scene, or "by the numbers" explainer.
- You ask to turn a pitch deck or slide outline into a narrated presentation video.
- You need one chart template rendered across many datasets for batch export.

## Example prompts

- "Turn this CSV of yearly market share into a bar chart race."
- "Animate a count-up from 0 to 1,287 with a thousands separator."
- "Make an animated 'how it works' infographic with 3 steps and connectors."
- "Convert this pitch deck into a narrated video where each slide tracks its voiceover."
- "Render this chart template once for every JSON file in my data folder."
